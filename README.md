# This App was built with Next.js

### Steps to start the project

### Pre requisites:

1. Install Docker
2. Install Docker Compose

### Start the project:

```bash
docker-compose up
# or
docker-compose up -d
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
